## Overview

Full Stack Javascript Multiplayer Tetris Game build with Node, socket.io, React and Redux.

Play it at: https://cepalle.github.io/red-tetris/

## Run it locally

After:
````cmd
 npm install
````
run:
````cmd
 npm run build
````
and:
````cmd
 npm run serve
````

RED TETRIS will be available at localhost:4433.
